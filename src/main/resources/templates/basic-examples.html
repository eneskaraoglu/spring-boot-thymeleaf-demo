Temel Thymeleaf Örnekleri

Bu sayfa Thymeleaf'in temel özelliklerini göstermek için hazırlanmıştır.

  1. Ana Sayfa
  2. Temel Örnekler

1. Metin İfadeleri

Thymeleaf, HTML içinde dinamik içerik göstermek için ${...} ifadesini kullanır.

<p th:text="${username}">Varsayılan Kullanıcı</p>
Sonuç

Varsayılan Kullanıcı

2. Koşullu İfadeler

Thymeleaf'te koşullu ifadeler için th:if, th:unless ve th:switch/th:case kullanılır.

<div th:if="${isAdmin}">
  Admin kullanıcısı içeriği
</div>
<div th:unless="${isAdmin}">
  Normal kullanıcı içeriği
</div>
Sonuç
Admin kullanıcısı içeriği görüntüleniyor.
Normal kullanıcı içeriği görüntüleniyor.

3. Döngüler (Iteration)

Liste ve koleksiyonlar üzerinde döngü yapmak için th:each kullanılır.

<ul>
  <li th:each="num : ${#numbers.sequence(1, 5)}" th:text="${num}"></li>
</ul>
Sonuç

4. Formatlama

Thymeleaf, tarih, saat ve sayılar için çeşitli formatlama özellikleri sunar.

<p>Tarih: <span th:text="${#dates.format(currentDate, 'dd MMMM yyyy')}"></span></p>
<p>Saat: <span th:text="${#dates.format(currentDate, 'HH:mm:ss')}"></span></p>
Sonuç

5. URL Oluşturma

Thymeleaf, @{...} sözdizimi ile URL'ler oluşturabilir ve parametreler ekleyebilir.

<a th:href="@{/users(id=1, action='view')}">Kullanıcı Detayı</a>
<a th:href="@{/users/1}">Kullanıcı Detayı (RESTful)</a>
Sonuç

Kullanıcı Detayı (Query Parametresi)

Kullanıcı Detayı (RESTful)

6. İfade Değişkenleri

Thymeleaf, *{...} sözdizimi ile seçili nesne içindeki özelliklere erişebilir.

<div th:object="${session.user}">
  <p>İsim: <span th:text="*{firstName}"></span></p>
  <p>Soyisim: <span th:text="*{lastName}"></span></p>
</div>
Sonuç

Bu örnek, bir HTTP oturumunda kullanıcı nesnesi olduğunda çalışır.
